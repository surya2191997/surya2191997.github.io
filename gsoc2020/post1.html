June 8rd, 2020

So this blog post concludes the official first week of coding period which began from June 1st. I have successfully implemented the header files for both the layers (CNN3D and MaxPool3D). Both the header files are within CNN_3D directory, within inc/DNN. At a high level, these header files contain the class constructor for the layers, as well as forward and backward pass functions, which call architecture specific functions within them. I also implemented ParseCNN3DLayer() and ParseMaxPool3DLayer(), which basically take in a string description of the layer, and construct the corresponding layer class object. Apart from this, I also implemented AddCNN3DLayer() and AddMaxPool3DLayer() which take in layer class object as parameter and add it to the overall deepnet.
