June 3rd, 2019

So this blog post concludes the first week of coding period which began from May 27th. My tasks this week as per the proposal were as follows:

  • Finalize class structure for the LSTM class
  • Implement forward pass for the LSTM layer

I have successfully completed both the tasks of this week. The class structure was implemented in a header file named LSTMLayer.h that I created in tmva/tmva/inc/TMVA/DNN/LSTM and the design was inspired from the file RNNLayer.h for the RNN. The cell forward pass was implemented as a method of the class in the header file itself, as is done for the RNN layer.
