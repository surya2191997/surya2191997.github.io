The past couples of week went into implementing the LSTM backward pass for cpu and ref architectures and also the tests for them. The backward pass function - LSTMLayerBackward() specific to an architecture is called from inside the Backward() function in LSTMLayer.h header file. The LSTMLayerBackward() function is implemented in the RecurrentPropagation.cxx file specific to each architecture type. For example, the cpu LSTMLayerBackward() function is inside the file tmva/tmva/src/DNN/Architectures/Cpu/RecurrentPropagation.cxx . The forward pass is implemented in the header file itself and there is no function specific to each architecture.

The corresponding tests are implemented within the /tmva/tmva/test/DNN/LSTM/ directory. There are separate(.cxx) files for cpu and ref and they include a common header file which implements the test.
